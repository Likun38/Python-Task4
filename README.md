

#  Python File Handling Projects

This repository contains two simple Python programs that demonstrate basic file handling operations such as reading, writing, appending, and handling errors.

---

##  Task 1 – Read a File and Handle Errors

**Description:**
This program reads a text file named `sample.txt` and displays its contents line by line. It also handles the case where the file does not exist using exception handling.

**Concepts Used:**

* File reading (`r` mode)
* Line-by-line iteration
* `try-except` block
* Handling `FileNotFoundError`

**How to Run:**

```bash
python task1.py
```

---

##  Task 2 – Write and Append Data to a File

**Description:**
This program takes user input, writes it to a file named `output.txt`, appends additional input to the same file, and then displays the final content of the file.

**Concepts Used:**

* File writing (`w` mode)
* File appending (`a` mode)
* File reading (`r` mode)
* User input handling

**How to Run:**

```bash
python task2.py
```

---

##  Requirements

* Python 3.x
